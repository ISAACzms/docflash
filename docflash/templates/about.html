About Doc Flash

Fast, intelligent document processing with the power of AI

What is Doc Flash?

Doc Flash is a powerful document intelligence platform that enables you to extract structured information from unstructured documents using advanced AI and OCR technologies.

Built on top of the excellent LangExtract framework, Doc Flash provides an intuitive web interface for:

  • Schema Definition: Define what information you want to extract
  • OCR Processing: Convert PDFs to text using advanced multimodal AI models
  • AI Extraction: Use large language models to extract structured data
  • Multi-Provider Support: Works with Azure OpenAI, OpenAI, Google Gemini, and custom models
  • Validation & Export: Validate results and export in multiple formats

Created By

Lionel Martis developed Doc Flash as a user-friendly interface to make document intelligence accessible to everyone. The tool builds upon the solid foundation provided by the LangExtract framework.

Getting Started

Ready to start extracting structured data from your documents? Here's how to get started:

  1. Configure your LLM provider in the .env file
  2. Define your schema - specify what information to extract
  3. Upload sample documents or use OCR to process PDFs
  4. Generate examples with AI assistance
  5. Run extraction and validate results
  6. Export and visualize your structured data
